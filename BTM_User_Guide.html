The simulator works exactly like a real ship's bridge. The Instructor sits at the Master Stations and controls the exercise. The Students operate the slave displays.

Step 3 — Master Station A (Ship Setup)

This is the Instructor's main setup screen. Configure everything here before starting the exercise.

Left Panel — Own Ship
1
Select Vessel Type
Click the SELECT VESSEL dropdown at the top of the left panel. Choose from 14 vessel types: Container Ship, VLCC Tanker, LNG Carrier, Bulk Carrier, Cruise, Ferry, Tug, Naval, Yacht, etc. All ship particulars (LOA, beam, draft, displacement, engine power, max speed) auto-fill automatically.
2
Set Ship Name, IMO, Call Sign
Type the vessel name in the VESSEL NAME field. This appears on ECDIS and AIS displays.
3
Set Loading Condition
In the Cargo & Loading card, select: Laden Full / Laden Partial / Ballast Normal / Light Ship. This affects draft and UKC calculations.
4
Set Initial Position & Heading
In the centre panel, enter the starting latitude, longitude, heading, and SOG. Default is Bosphorus Strait entrance. You can type any position in degrees/minutes format.
Centre Tabs — Route, Weather, Failures
5
Load a Route (Route & Waypoints tab)
Click the ROUTE & WAYPOINTS tab. Click the preset dropdown and select a route — Bosphorus Northbound, Dover Strait, Singapore, etc. Waypoints auto-load. You can also add custom waypoints by clicking + ADD WPT.
6
Set Weather & Environment
Click the WEATHER & ENVIRONMENT tab. Adjust:
• Beaufort Scale — select 0-12, wind speed auto-sets
• Wave Height & Period — affects 3D visual rolling
• Current Rate & Direction — affects ECDIS drift
• Visibility — affects 3D fog and radar
• Time of Day — Dawn/Day/Dusk/Night lighting
• Fog toggle — instantly sets visibility to 200m
7
Set Up Equipment Failures (Failures tab)
Click the FAILURES & HAZARDS tab. Check any boxes to pre-arm failures: Gyro failure, GPS failure, Radar failure, Main Engine failure, Steering failure, Power Blackout, Fire, Flooding, MOB. Click TRIGGER SELECTED FAILURES at any point during the exercise to inject them.
Right Panel — Target Ships
8
Add Target Ships
In the right panel under ADD TARGET:
1. Select vessel type (Tanker, Container, Bulk, Ferry, Fishing, Naval, Tug, etc.)
2. Set Bearing from own ship (0-359°)
3. Set Range in NM (e.g. 6.0)
4. Set Target Course (0-359°)
5. Set Target Speed in knots
6. Click + ADD TARGET
Repeat for up to 20 targets. All appear on Radar and ECDIS instantly.
Engine Telegraph
9
Set Engine Order
The telegraph is in the right panel. Click any position: FULL AHEAD → HALF AHEAD → SLOW AHEAD → DEAD SLOW AHEAD → STOP → DEAD SLOW ASTERN → SLOW ASTERN → HALF ASTERN → FULL ASTERN. The Conning Display updates instantly showing RPM change.
10
Apply All Settings
Click the green ▶ APPLY ALL button at the top of the centre panel. This broadcasts everything to all slave stations.
The live digital displays (SOG, COG, HDG, Depth, UKC) at the top of the centre panel update every second automatically. Watch UKC — it turns amber below 5m and red below 3m.
Step 4 — Master Station B (Simulator Commander)

This is the Exercise Control Panel. Start, stop, pause and record exercises from here.

1
Load a Scenario
In the Scenario Library (left panel), browse 15 built-in exercises. Filter by area (Bosphorus, Dover Strait, Singapore, etc.) or difficulty (Easy/Medium/Hard). Click a scenario to select it. The right panel shows description and duration.
2
Add Trainees
In the right panel under Student Groups, type each student's name and click + Add Trainee. Their names appear in the assessment report.
3
Launch All Stations
Click ▶ LAUNCH ALL in the Station Status panel. This sends a signal to all open tabs. Their LEDs turn green one by one as they respond.
4
Start the Exercise
Click the large green ▶ START button in the centre. The simulation clock starts counting. All stations receive the start signal and begin updating.
5
Control Simulation Speed
Use the SPEED slider to run at ×0.25 (slow motion), ×1 (real time), ×2, ×4, or ×8 (fast forward). Useful for fast-forwarding through long transits.
6
Inject Emergency Scenarios
The Quick Inject panel on the right lets you inject situations mid-exercise with one click:
• 🌫 FOG — drops visibility to 200m instantly
• ⛈ STORM — Beaufort 9 with heavy seas
• ⚙ MAIN ENGINE FAIL — engine stops
• 🛞 STEERING GEAR FAIL — rudder jams
• 👤 MAN OVERBOARD — MOB alarm on all stations
• 💡 POWER BLACKOUT — all systems alarm
7
Pause / Resume
Click ⏸ PAUSE to freeze all stations for discussion. Click ▶▶ RESUME to continue. Use ⚡ FREEZE for instant pause without confirmation.
8
Record & Debrief
Click ⏺ START RECORDING before the exercise. After stopping, click 💾 SAVE. Then click 📊 EXPORT DEBRIEF to download an HTML report with scores, event log, and timestamps — ready to print or email to students.

Full professional radar display. Operates like a real JRC/Furuno/Kongsberg ARPA radar.

Acquiring Targets (ARPA)
1
Manual Acquisition — Click on Echo
Move your mouse over any echo on the radar screen and left-click it. A triangle ARPA symbol appears. The target data panel opens on the right showing BRG, RNG, CPA, TCPA, BCR, BCT, name and MMSI.
2
Automatic Acquisition
Click AUTO ACQ in the ARPA panel. All targets within radar range are automatically acquired. Use with caution — it acquires everything including buoys and land.
3
Read CPA / TCPA
After acquiring a target, check the target list on the right panel. CPA values show in green (safe), amber (caution), or red (alarm). When CPA is below the CPA LIMIT and TCPA is below TCPA LIMIT, an alarm triggers automatically.
EBL and VRM
4
Using Electronic Bearing Line (EBL)
Click EBL1 OFF to activate EBL1 (amber dashed line from centre). Type bearing in the EBL1 input box (e.g. 045 for 045°T). The line extends to the edge of the screen. Use to measure bearing to a target or landmark.
5
Using Variable Range Marker (VRM)
Click VRM1 OFF to activate VRM1 (amber dashed circle). Type range in NM in the VRM1 input. The circle shows that range. Use to measure distance to a target or plan CPA.
6
Guard Zone
Set INNER and OUTER range in the Guard Zone card. Click GUARD OFF to activate. Any target entering the zone triggers an alarm.
7
Trial Manoeuvre
Click T-SIM OFF to activate trial manoeuvre. Enter a trial COG and SOG. A dashed cyan vector shows where own ship would go. Check if this avoids all CPA alarms before executing.
Hover the mouse over the radar screen to see cursor BRG and RNG in the bottom bar. This is how you measure bearing and range to any point on screen without acquiring it.
Step 6 — Conning Display

The bridge instrument panel — gyro compass rose, engine telegraph, rudder angle indicator, and autopilot all update in real time.

InstrumentReadingAction
Gyro Compass RoseRotating compass card shows heading. Red lubber line at top = ship's head. Green dashed line = set course.Watch heading change as autopilot steers. Gyro 1 and Gyro 2 toggles simulate failure.
Engine TelegraphLit position = current order. RPM bar shows actual engine speed building up.Click any position on the right panel to give telegraph order.
Rudder Angle IndicatorCyan needle = actual rudder. Amber dashed = ordered rudder. PORT (red left), STBD (blue right).Use the rudder slider on right panel or click 35P / MDSHIP / 35S quick buttons.
AutopilotShows mode (HAND/AUTO/NFU), set course, actual heading, deviation, ROT.Click HAND/AUTO/NFU buttons. In AUTO, type set course and autopilot steers to it.
Depth / UKCDepth below surface and below keel shown in left panel.Set shallow water alarm value. Red alarm if UKC drops below 2m.
GPS PositionLat/Lon with XTE — Cross Track Error from planned route.Watch XTE: if it exceeds 0.1NM an alarm fires and banner shows PORT or STBD of track.
Bow Thruster: Enable with the toggle then slide power left (PORT) or right (STBD). Used for berthing manoeuvres at slow speed. Has no effect at speeds above 4 knots.
Step 7 — ECDIS (Electronic Chart)

The electronic chart showing the Bosphorus Strait with own ship, AIS targets, route, depths, and dangers.

1
Navigate the Chart
• Zoom in — scroll mouse wheel up, or click ZOOM + button
• Zoom out — scroll mouse wheel down, or click ZOOM -
• Pan / Move chart — click and drag with left mouse button
• Quick scale — use the Quick Scale dropdown (1:500,000 to 1:12,500)
• FIT button — recentres chart on own ship position
2
Read Chart Information
• Depth contours — coloured lines. Amber = safety contour. Red = shallow/danger.
• Depth soundings — numbers on chart in metres. Red = below safety depth.
• Dangers / Rocks — red cross symbols with depth value
• Buoys — diamond shapes: green = starboard, red = port, amber = special
• TSS lanes — blue dashed lines showing traffic separation
• Own ship — green silhouette with heading line and speed vector
• AIS targets — cyan triangles pointing in direction of travel
3
Turn Display Layers On/Off
Left panel has toggles for every chart layer. Toggle off Depth Soundings for a cleaner chart. Toggle off TSS/VTS Zones if not needed. Toggle AIS Targets to hide/show all traffic.
4
Set Safety Settings
In the Safety Settings card (left panel), set:
• Safety Depth — soundings shallower than this show in red
• Safety Contour — the bold amber depth contour line
• Shallow Contour — very shallow water warning
• Bridge Height — for aerial clearance checks
5
Load a Route
Right panel → Route Planning → click the preset dropdown → select a route. Waypoints appear as green circles on the chart connected by a dashed green line. A 0.1NM XTE corridor is shown.
6
Add Custom Waypoints by Clicking
Click + ADD WPT. The button changes to say "CLICK CHART: ADD WPT". Now click anywhere on the chart to add a waypoint at that position. Click again for the next waypoint. Click the button again to stop adding.
7
AIS Target Information
Hover your mouse over any cyan triangle (AIS target) to see a popup showing: Name, Type, BRG, RNG, COG, SOG, CPA, TCPA. The AIS list on the right is sortable by distance or CPA — red items are close approach risks.
8
Mark MOB Position
Click MOB ON CHART in the left panel. A pulsing red MOB marker appears at own ship's current position. An alarm fires on all stations. Use this during MOB exercise scenarios.

Troubleshooting
Radar Range Buttons Not Working
✓
Fixed in latest version
Download the latest BTM_Radar_XBand_ARPA.html from your Claude conversation. Range buttons now correctly highlight and the ring scale updates on click.
Stations Not Communicating
All files MUST be in the same folder. Files from different folders cannot communicate.
1
Check they are all in one folder
Open Windows Explorer → navigate to BTM_Simulator folder → confirm all 6 files are there.
2
Check they are all open in SAME Chrome window
All 6 tabs must be in the same Chrome window, not different windows. BroadcastChannel only works within the same browser origin.
3
Hard refresh all tabs
Click each tab → Ctrl + Shift + R. Do this for all 6 tabs in sequence.
3D Visual Black Screen
1
Enable hardware acceleration
chrome://settings/system → Use hardware acceleration: ON → Relaunch Chrome
2
Check WebGL is working
chrome://gpu ← look for "WebGL: Hardware accelerated"
Console Error — Any Module
1
Get the exact error message
Press F12 → click Console tab → copy the red error text → paste it into the Claude conversation. The error will be fixed immediately.
Display Too Small
1
Use F11 for full screen
Press F11 on any tab for true full-screen with no browser chrome. Press F11 again to exit.
2
Reduce browser zoom if needed
Press Ctrl + Minus (-) to zoom out Chrome. Ctrl + 0 to reset to 100%.
For the best classroom experience: Use F11 on the Radar and ECDIS tabs. These are now wider with narrower panels as of the latest update.
