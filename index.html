HOW TO USE THIS PAGE
1
Open Chrome
Use Google Chrome only. Other browsers do not support all simulator features.
2
Click a Module Button
Each coloured button opens that module in a new tab. Students open Radar, ECDIS, Conning, or 3D Visual.
3
Press F11 for Full Screen
After the module opens, press F11 on your keyboard for a full-screen professional display.
4
Instructor Opens Master A + B
The instructor opens both Master Station A and B. These control the entire simulation.
5
All Modules on Same Computer Sync
Open multiple tabs in the same Chrome window — they communicate automatically. No setup needed.
6
Read the User Guide
Open the User Guide for complete step-by-step operating instructions for every module.
